# OpenDeploy

OpenDeploy helps Cursor deploy and operate projects through the official `@opendeploydev/cli`.

## Usage

Invoke the skill from Cursor Agent chat:

```text
/opendeploy deploy this project.
# or
/od deploy this project.
```

The canonical entrypoint is `opendeploy`; `deploy` and `od` are short aliases. The plugin also ships setup, auth, context, config, env, database, domain, debug, ops, alarms, oncall, AI Hub, and API skills.
